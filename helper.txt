# 1. File -> New Project (указываете название, указываете версию Pyhton)
2. Создаёте файл с зависимостями requirements.txt
3. Настраиваете виртуальное окружение File -> Settings -> Project -> Python Interpreter -> Add...
4. Устанавливаете зависимости при помощи pip install -r путь_до_файла
5. Создаёте Django-проект django-admin startproject название .
6. Создаёте базу данных
7. Создаёте приложения python3 manage.py startapp название
8. Настройки проекта: настройки БД, зарегистрировать приложения в INSTALLED_APPS
9. Проверить python3 manage.py runserver и пройти по ссылке





Развернуть проект с приложением product:
Создать модель Product c полями: title, description, price и зарегистрировать его в админ панели.
Создать записи в БД
Создать вьюшки 3 способами (функцией, наследуясь от APIView и при помощи GenericView):
1. Список продуктов
2. Детали продуктов
3. Создание продуктов
4. Редактирование
5. Удаление
Затем переписать все вьюшки с использованием ViewSet
Залить проект в Github
